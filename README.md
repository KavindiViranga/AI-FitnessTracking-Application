🏋️ AI Fitness Tracker

An AI-powered full-stack fitness web application that allows users to set daily fitness goals, track calorie intake, log workouts, and analyze food images to automatically estimate calorie consumption using Google Gemini AI.

This project demonstrates modern full-stack development combined with AI integration.

🚀 Features

✅ Set daily fitness goals

✅ Track calories consumed

✅ Track calories burned from workouts

✅ User authentication (Sign Up / Sign In)

✅ Update and manage user profile

✅ Upload food images for calorie estimation

✅ AI-powered food image analysis using Google Gemini AI

🛠️ Tech Stack

React JS – Frontend

Strapi – Headless CMS & Backend API

Tailwind CSS – UI Styling

Google Gemini AI – Food Image Analysis

Strapi Cloud & Vercel – Deployment

⚙️ How It Works

Users create an account and set daily fitness goals.

Users log food intake and workout activities.

For food tracking, users upload an image of their meal.

The image is sent to Google Gemini AI.

The AI analyzes the image and estimates calorie information.

The system updates total calories consumed and burned.
